# GeminiChatBot

A simple AI chatbot app built with SwiftUI and Firebase AI that lets users chat with Google's Gemini model.

## Features

- Chat with Gemini AI
- View conversation history during the current session
- Clear chat history
- Loading indicator while waiting for responses
- SwiftUI-based interface

## Built With

- Swift
- SwiftUI
- Firebase AI
- Google Gemini

## Setup

1. Clone the repository.
2. Add your `GoogleService-Info.plist` file from Firebase.
3. Open the project in Xcode and run the app.
